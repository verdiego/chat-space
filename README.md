## users table

|Column|Type|Options|
|------|----|-------|
|name|string|index: true, null: false, unique: true|
|mail|string|null: false|

### Association
- has_many :messages
- has_many :members


## groups table

|Column|Type|Options|
|------|----|-------|
|name|string|null: false, unique: true|

### Association

|Column|Type|Options|
|------|----|-------|
|content|string||
|image|string||
